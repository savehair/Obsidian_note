根据提供的资料，多元统计分析主要包含**数据降维**、**分类判别**、**关系分析**和**综合评价**四大类算法。以下是各核心算法的基本概念与思想总结：

# 1 聚类分析 (Cluster Analysis)

**基本概念：** 是研究“物以类聚”的一种统计分析方法，其本质在于建立分类。对样品的聚类称为Q型聚类，对变量的聚类称为R型聚类。
**核心思想：**
- **物以类聚**：在样品距离的基础上定义类与类之间的距离。首先将 n 个样品分成 n 类,每个样品自成一类。 然后每次将具有最小距离的两类合并，合并后重新计算类与类之间的距离,这个过程一直持续到所有的样品归为一类为止。
- **常用方法：**
    - **系统聚类法 (Hierarchical Clustering)**：首先将 $n$ 个样品各自看作一类，然后每次合并距离最小（最相似）的两类，并重新计算新类与其他类的距离，重复此过程直到所有样品归为一类,。
        - [[计算#1.1 系统聚类法 (Hierarchical Clustering)|计算方法]]
    - **快速聚类法 (K-Means)**：任取 $k$ 个初始中心，将每个样品归入最近的中心所在的类，然后重新计算每类的中心（均值），迭代直到分类不再变化。
        - [[计算#1.2 快速聚类法 (K-Means)|计算方法]]


# 2 主成分分析 (Principal Component Analysis, PCA)

**基本概念：** 是将多指标化为少数几个综合指标的一种统计分析方法。PCA通过*降维技术*把多个变量化为几个少数主成分，这些主成分保留原始变量的*绝大部分信息*，通常表示为*原始变量的线性组合*。
**核心思想：**
- **降维与保真**：将原来众多具有一定相关性的指标重新组合，生成一组新的、相互无关的综合指标。这些新指标（主成分）按方差大小排序，前几个主成分尽可能多地保留了原始数据的信息（方差）。
- **计算逻辑**：对协方差矩阵或相关系数矩阵进行特征分解，利用特征值和特征向量确定载荷矩阵。通常根据累计方差贡献率（如>80%）来确定保留的主成分个数,。

# 3 因子分析 (Factor Analysis)

**基本概念：** 因子分析是主成分分析的推广，旨在用少数几个潜在的、不可观测的公共因子来解释多个观测变量之间的相关关系。
**核心思想：**
- **潜在结构探究**：认为可观测变量是由少数几个“公共因子”和“特殊因子”共同决定的。它不仅进行降维，更注重解释变量背后的实际含义（如通过因子旋转技术使因子更具解释性）,。

**与PCA的区别**：
- 主成分分析是将多指标化为少数几个综合指标的一种统计分析方法，因子分析是主成分分析的推广和发展,它是研究相关系数矩阵或协方差矩阵的内部依赖关系，将多个变量综合为少数几个因子。主成分分析主要用于数据降维、理解数据结构和去除数据中的冗余信息，因子分析主要用于确定变量之间的潜在结构和关系，进行数据降维和特征提取。
- 相同之处：都是用较少的综合变量来代替原来较多的变量，这几个变量尽可能多地反映原来变量的信息，并且彼此之间不相关，本质上都使用了降维的思想。
- 不同之处：
	1. 主成分分析是进行变量变换，因子分析需要构造因子模型，并赋予因子实际含义；
	2. 主成分分析没有模型假设，而因子分析假设各公共因子方差为 1，公共因子之间不相关、特殊因子之间不相关、公共因子与特殊因子不相关；
	3. 因子分析可以使用主成分法、极大似然法等进行因子提取，而主成分分析只能基于相关系数矩阵或协方差矩阵进行分解计算；
	4. 因子分析结果会随选取因子数量发生变化，而主成分分析结果不随选取的主成分个数变化；
	5. 因子分析可以使用旋转技术，而主成分分析不使用。

# 4 判别分析 (Discriminant Analysis)

**基本概念：** 在已知研究对象已经被分成若干类的情况下，建立判别函数，确定新的样本属于已知类别中的哪一类。给出一个衡量新样品与各已知类别接近程度的描述统计模型，即判别函数，同时还要指定一种判别规则，借以判定新的样本的归属
**核心思想与方法：**

- **距离判别**：计算新样本到各个类别中心的距离（如马氏距离），按“距离最近”原则归类。
- **Fisher判别**：构造线性判别函数，使得不同类别之间的离差尽可能大（分得开），而同一类别内部的离差尽可能小（聚得拢）。
	- [[计算#6.1 Fisher 判别 (线性判别)|计算方法]]
- **Bayes判别**：引入概率思想，综合考虑各总体的先验概率和错判损失，计算样本属于各类的后验概率，选择概率最大的一类,。
	- [[计算#6.2 Bayes 判别 (概率判别)|计算方法]]

# 5 对应分析 (Correspondence Analysis)

**基本概念：** 对应分析通过降维技术，将R型（变量）因子分析和Q型（样品）因子分析结合起来，在一个二维图形上同时展示变量和样品的关系,。
**核心思想：**
- **同构映射**：解决了传统因子分析中变量和样品分离处理造成的信息损失。它通过数学变换，将行变量（样品）和列变量（指标）映射到相同的坐标轴上，直观地揭示它们之间的内在联系（如哪个样品在哪个指标上表现突出）。
# 6 典型相关分析 (Canonical Correlation Analysis)

**基本概念：** 研究**两组**变量之间整体相关性的统计方法。
**核心思想：**
- **组间最大相关**：不直接分析两组变量中单个变量的一一对应关系，而是分别寻找两组变量的线性组合（典型变量），使得这两组线性组合之间的相关系数（典型相关系数）达到最大。这其实是主成分分析思想在两组变量关系研究中的推广,。
[[计算#5 典型相关分析 (CCA)|计算方法]]
# 7 综合评价 (Comprehensive Evaluation)

**基本概念：** 当评价对象涉及多个指标时，将这些指标转化为一个能够反映整体情况的综合指标进行评判。
**核心思想：**
- **赋权与融合**：核心步骤是确定各指标的权重（如使用层次分析法、PCA、熵值法等），然后通过加权平均等方式将无量纲化处理后的数据合成一个总分，用于排序或分级。
[[计算#2 综合评价 (Comprehensive Evaluation) - 层次分析法 (AHP)|计算方法]]

**总结建议：** 在学习这些算法的计算题目时，重点关注以下通用流程：**数据标准化** $\rightarrow$ **计算相关/协方差矩阵** $\rightarrow$ **求特征值与特征向量** $\rightarrow$ **构造函数或得分模型**。
