# 1 第1章 管理导论
**【提纲重点 1】管理的基本职能 (必须要背)**
- **教材原义**：管理工作包含五大职能，它们构成了管理过程的逻辑闭环。
    1. **决策**：**“管理的核心”**。识别机会与威胁，选择活动方向和内容。
    2. **组织**：设计结构、人员配备、权力分配，保障决策实施。
    3. **领导**：指挥、影响、激励组织成员，解决“人”的问题。
    4. **控制**：监督、检查、纠偏，确保实际与计划相符。
    5. **创新**：**“管理的灵魂”**。在变动的环境中进行变革和调整，维持组织生存与发展。

**【提纲重点 2】管理的基本工具 (必须要背)** 管理者通过什么手段来影响他人？主要有两种工具：
1. **权力 (Power)**：
    - 包括**制度权力**（由职位赋予，如强制权、奖赏权）和**个人权力**（由个人魅力、专长产生）。
    - _背诵要点_：权力是单向的（命令与服从），也是互动的（影响与被影响）。
2. **组织文化 (Organizational Culture)**：
    - 一种“软”工具，通过共同的价值观和行为准则引导成员行为。

**【提纲重点 3】效率与效果 (必须要背)** 这是管理的两大核心评价指标：
- **效果 (Effectiveness)**：**“做正确的事”** (Do the right things)。指目标的达成程度。
- **效率 (Efficiency)**：**“正确地做事”** (Do things right)。指资源利用的产出/投入比（低浪费）。
- _关系_：管理既要追求高效率，也要追求高效果，通常二者是相辅相成的。

*【章节基础补充 】*
- **管理的本质**：
	- 是对人或对人的行为进行**协调和管理^[管理者对事的管理的目的是通过对人的管理来实现的，其主要工作是选择对的人去做对的事情]**。
- **管理属性**：既有**科学性**（理论、规律、程序），又有**艺术性**（实践中的权变、创造性）。
- 管理者应该**具备的技能**：*理念技能、人际关系技能，专业技能、政治技能。*
- **管理的基本工具**
	- 权力
	- 组织文化：其核心是组织成员的*普遍认同、共同接受*的价值观以及价值观所决定的行为准则
# 2 第2章 管理理论的历史演变
## 2.1 古典管理
主要标志是：==泰勒的科学管理原理、法约尔的工业管理与一般管理==
**【提纲重点 4】泰勒科学管理理论的主要内容 (必须要背)** 泰勒被称为“科学管理之父”，其理论重点在于提高**劳动生产率**。
- **核心内容**：
    1. **改进工作方法（标准化）**：用科学的动作研究替代经验判断，制定标准化的操作方法和工具（如铁锹实验）。
    2. **作业环境与条件标准化**：确保工具、环境标准化。
    3. **挑选和培训工人**：让“一流的工人”做适合的工作。
    4. **差别计件工资制**：完成定额给高工资，完不成给低工资，刺激积极性。
    5. **职能工长制**：对管理工作细分，每个管理者只能承担一两项工作。
    6. **设置计划部门**：将计划职能（管理层）与执行职能（工人）分开。
    7. **例外原则**：高层只管例外事项或者重要问题，日常事务授权下级。
- **核心思想**：一场“心理革命”，劳资双方从对立转向合作，共同把蛋糕做大。

**【提纲重点 19】X理论与Y理论 (必须要背)** _注：虽然麦格雷戈属于行为科学学派（第2章提及），但具体内容常结合第9章领导或第10章激励考查，此处先做理论铺垫。_

- **X理论**（人性本恶）：认为人天生懒惰、逃避责任，管理需要强制和惩罚。
- **Y理论**（人性本善）：认为人愿意工作、能自我管理，管理需要授权和激励。

**【章节基础补充 (理解即可)】** 为了知识完整性，你需要大致了解与泰勒同时代的另外两位大师（形成“古典管理理论”）：

- **法约尔（一般管理理论）**：提出了管理的**14条原则**（如统一指挥、等级链）和**5大要素**（计划、组织、指挥、协调、控制）。
- **韦伯（科层组织理论）**：被称为“组织理论之父”。提出了"**理性-合法**"权力^[有传统型权力、个人魅力型权力、法理型权力]，主张建立非人格化、层级分明、照章办事的理想行政组织体系。
	- 其主张法理型权力，因为有下述特征
		- 为*连续性*奠定了基础。权力是给职位的不是给人的
		- *合理性*。根据个人能力挑选
		- 领导者可以借助法理手段保证权力
		- 权力有明确规定，以组织任务来划分
