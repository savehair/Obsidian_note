# 1 **第3章 决策与决策过程**
**【提纲重点 5】决策的分类 (必须要背)** 教材根据不同维度对决策进行了划分，考试常考选择题或名词辨析：
1. **按环境可控程度**：
    - **确定型决策**：信息完全，后果确定。
    - **风险型决策**：信息不完全，但知道各种后果发生的**概率**，能*预知*备选方案及其结果发生的*可能性*。
    - **不确定型决策**：信息缺失，连概率都不知道（需用“小中取大”、“大中取大”等原则）。
2. **按问题重复程度**：
    - **程序化决策**：针对例行、重复的问题（如请假制度、采购流程）。
    - **非程序化决策**：针对例外、新颖、重大的问题（如新产品开发、组织变革）。
3. **按决策主体**：
    - **个体决策**：效率高，但易受个人局限。
    - **群体决策**：*集思广益*，不仅能得到更多的方案，还容易得到认可。但效率低，会被*个体的影响*，即便使人更可能承担责任，同时也可能因此导致“*责任分散*”。

**【提纲重点 6】决策过程 (必须要背)** 西蒙认为“管理就是决策”。教材将决策过程分为6个步骤,：
1. **识别问题**：发现现状与预期的差距（起步）。
2. **诊断原因**：分析导致问题产生的根本原因。
3. **确定目标**：明确要达成什么结果。
4. **制定备选方案**：设计多个可行方案（不要只盯着一个）。
5. **评价、选择方案**：依据标准^[可行性、有效行、满意程度]进行取舍。
6. **实施和监督**：将方案转化为行动并进行反馈。

**【教材深度补充 (理解重点)】**

- **决策的原则**：教材强调决策遵循的是“**满意原则**”（西蒙提出，有限理性），而不是“最优原则”（古典决策理论，完全理性）。管理者只能在有限信息下做出“令人满意”的选择,。

# 2 **第4章 环境分析与理性决策**
这一章提供了决策所需的“弹药”（环境分析工具）。
**【提纲重点 7】组织内外部环境要素 (框架要熟)**

1. **外部环境**：
    - **宏观环境 (一般环境)**：**PEST分析**——政治(Politics)、经济(Economy)、社会(Society)、技术(Technology)、自然资源。
    - **微观环境 (具体环境)**：直接影响企业的因素，包括顾客、供应商、竞争者、管制机构等,。
2. **内部环境**：组织拥有的资源（人、财、物）和组织文化。

**【提纲重点 8】五力分析模型与SWOT分析 (必须要背 + 会应用)** 这是案例分析题的万能钥匙。

1. **五力模型 (迈克尔·波特)**： 用于分析**行业竞争环境**。
    - (1) 现有企业间的竞争（来自行业竞争者。
    - (2) 潜在进入者的威胁。
    - (3) 替代品的威胁（来自其生产者。
    - (4) 供应商的讨价还价能力（来自卖方。
    - (5) 买方（顾客）的讨价还价能力。
2. **SWOT分析**： 用于综合分析内外部环境。
    - **S** (Strengths) 优势、**W** (Weaknesses) 劣势 —— **内部因素**。
    - **O** (Opportunities) 机会、**T** (Threats) 威胁 —— **外部因素**。
    - _应用_：通过组合形成SO（增长型）、ST（多元化）、WO（扭转）、WT（防御）战略。

**理性决策** 是管理学中一个理想化的理论模型，通常被称为“科学决策”。
我们可以从以下三个维度来归纳：
1. **核心假设**：**完全理性 (The "Economic Man")** 理性决策建立在“经济人”假设之上，认为决策者是完全理性的，其唯一的目的是**实现组织利益最大化**。
2. **理想化的前提条件 (现实中很难达到)** 要实现绝对的理性决策，必须满足以下严苛条件：
	- **信息完全**：决策者掌握关于问题的所有信息。
	- **目标明确**：组织对其要达到的目标有清晰、一致的认识。
	- **方案穷尽**：决策者能列出所有可能的备选方案，并预知每个方案的后果。
	- **偏好恒定**：决策标准和喜好是稳定不变的。
	- **无资源限制**：不受时间、成本等资源的约束。

3. **现实修正：有限理性 (Bounded Rationality)** 教材特别指出，完全理性在现实中几乎不存在。诺贝尔奖得主**西蒙 (Herbert Simon)** 对此进行了修正，提出了更符合现实的观点：
	- **有限理性**：受限于认知能力、信息不全和环境复杂性，人只能在一定范围内保持理性。
	- **满意原则**：决策者追求的不是“最优解” (Optimal)，而是“**满意解**” (Satisfying)——即足够好、能解决问题的方案。

---

**学习建议**： 在考试或案例分析中，如果题目问“为什么该经理没能做出最好的决定？”，通常就是用**有限理性**和**信息不对称**来解释；如果题目让你“按理性决策步骤分析”，则需要严格按照“识别问题 -> 确定标准 -> 拟定方案 -> 评价方案 -> 选择方案”的逻辑作答。

# 3 **第5章 决策的实施与调整（计划）**

决策定了，就要通过“计划”来落地。

**【提纲重点 9】计划编制过程 (逻辑要清)** 教材将计划编制分为以下逻辑步骤,：

1. **制定计划目标**：明确从哪里到哪里。
2. **估量现状与目标之间的差距**。
3. **预测未来情况**：分析未来环境变化。
4. **制定计划方案**：提出具体行动路线。
5. **实施和总结计划方案**。

**【提纲重点 10】目标管理的基本思想 (MBO) (核心考点)** 由**德鲁克**提出，是一种系统性的管理方法。

1. **核心思想**：
    - **以目标为中心**：用目标来统一员工意志。
    - **自我控制** (Self-control)：强调员工自我管理，而不是靠上级指令。
    - **参与式管理**：上下级共同协商制定目标。
2. **过程**：
    - 目标制定（自上而下分解） -> 目标实施（自我控制） -> 成果评价（奖惩依据）。

**【教材深度补充 (理解重点)】**

- **PDCA循环**：计划实施的通用模型。**P**(Plan 计划) -> **D**(Do 实施) -> **C**(Check 检查) -> **A**(Action 处理/纠偏)。大环套小环，螺旋上升。
