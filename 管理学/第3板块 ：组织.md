# 1 **第6章 组织设计 (重中之重)**

**【提纲重点 11】组织设计的任务和影响因素 (必须要背)**

1. **任务**：提供**组织结构设计**（静态）和**组织运行制度设计**（动态）。
	1. 组织结构设计：职能设计、部门设计、层级设计
	2. 组织运行制度设计：沟通系统设计、
2. **5大影响因素** (简答题常客)：
    - **环境**：环境复杂/变动大 $\rightarrow$ 柔性/有机式结构；环境稳定 $\rightarrow$ 刚性/机械式结构。
    - **战略**：结构服从战略（钱德勒提出）。如“单一产品战略”对应“职能制”，“多角化战略”对应“事业部制”。
    - **技术**：单件小批量、大批量、流程生产对结构要求不同（伍德沃德的研究）。
    - **规模**：规模越大，分工越细，规范化程度越高。
    - **生命周期**：创业阶段、集合阶段、规范化阶段、精细阶段，结构随之演变。

**【提纲重点 15】管理幅度与层级 (逻辑要清)**

1. **管理幅度 (Span of Control)**：一名管理者**直接**管理的下属数量。
2. **幅度与层级的关系**：在组织规模一定时，二者呈**反比**关系。
    - **扁平结构 (Flat)**：幅度大、层级少。_优点_：信息传递快、失真少、利于发挥下属积极性；_缺点_：主管负担重、难以严密控制。
    - **锥形结构 (Tall)**：幅度小、层级多。_优缺点_与扁平结构相反。

**【提纲重点 13】组织结构的形式 (必须要背 + 会画图)** 你需要掌握以下四种核心结构的**定义**、**优缺点**及**适用场景**：

1. **直线职能制 (Line-Functional)**：
    - _特点_：结合了“直线制”（统一指挥）和“职能制”（专业分工）的优点。
    - _优点_：分工细致、指挥统一、秩序井然。
    - _缺点_：部门间横向沟通差、适应环境变化能力差（僵化）。
    - _适用_：中小企业、环境稳定的组织。
2. **事业部制 (Divisional / M-form)**：
    - _核心原则_：**“集中政策，分散经营”**。
    - _特点_：按产品、地区或客户划分独立经营单位，独立核算，自负盈亏。
    - _优点_：有利于高层摆脱日常事务专心战略、有利于培养通才、适应性强。
    - _缺点_：机构重叠资源浪费、易产生本位主义（忽略整体利益）。
    - _适用_：大型、多角化经营的跨国企业。
3. **矩阵制 (Matrix)**：
    - _特点_：**双重指挥链**。既有垂直的职能部门，又有横向的项目小组。
    - _优点_：机动灵活、促进部门间协作、资源共享。
    - _缺点_：**“多头指挥”**（违背统一指挥原则），易导致混乱和推诿。
    - _适用_：需要跨部门协作的临时性、复杂项目（如研发、工程）。
4. **直线制**：最早期的形式，只有垂直领导，没有职能部门，仅适用于微型企业。

**【提纲重点 12】部门化与职权划分** 

1. **部门化 (Departmentalization)**：这是将组织中的工作任务和人员组合成若干管理单位（部门）的过程。简单说就是“归类打包”。
	**常见的部门化方式 (必须要背)**：
	1.  **职能部门化**：按工作职能分类（如研发部、营销部、财务部）。*最常见，有利于专业化，但易产生部门墙*,。
	2.  **产品部门化**：按产品线分类（如手机事业部、家电事业部）。*有利于对产品结果负责*,。
	3.  **地域部门化**：按地理区域分类（如华东区、华南区）。*适应当地市场*。
	4.  **顾客部门化**：按客户类型分类（如企业客户部、个人客户部）。
 2. 职权划分 (Division of Authority)
	在组织中，管理者手中的权力性质是不同的，教材将其分为三类：
	1.  **直线职权 (Line Authority)**：
	    *   **定义**：指管理者直接领导下属工作的权力。
	    *   *特点*：是**“指挥与命令”**的关系，沿着指挥链从上至下延伸（像一条直线）。
	    *   *核心*：拥有直线职权的人是“直线人员”，对组织目标的实现负直接责任（如总经理、车间主任）。
	2.  **参谋职权 (Staff Authority)**：
	    *   **定义**：指管理者向直线人员提供咨询、建议或服务的权力。
	    *   *特点*：是**“建议与服务”**的关系（不是命令）。
	    *   *核心*：参谋人员（如法务顾问、人力资源顾问）利用专业知识辅助直线人员决策。
	3.  **职能职权 (Functional Authority)**：
	    *   **定义**：指参谋人员在特定领域内，被授权直接指挥直线人员的权力。
	    *   *特点*：是**“有限的指挥权”**。
	    *   *例子*：财务总监（参谋）有权要求生产经理（直线）必须按规定填报财务报表。

**【重点难点：直线与参谋的矛盾】**,
*   **常见问题**：直线人员可能轻视参谋建议；参谋人员可能越权指挥。
*   **解决**：明确职权关系，直线要善于听取建议，参谋要懂“到位不越位”。


**【提纲重点 16】集权、分权与授权 (核心概念)**

1. **集权与分权**：
    - 不是绝对的，而是程度问题。
    - _判断标准_：决策权在较低管理层级的**数量**越多、**重要性**越大、**审核**越少，分权程度越高。
2. **授权 (Delegation)**：
    - 上级将权力委派给下级，但**“授责不授责”**（最终责任仍在管理者）。
    - _原则_：信任原则、目的性原则、权责一致原则。

**【提纲重点 14】组织结构的演变趋势 (了解)**

- **扁平化**（层级减少）、**柔性化**（适应变化）、**无边界化**、**虚拟化**-。

# 2 **第8章 组织文化**

虽然提纲未详细列出此章考点，但它是管理“软实力”的核心，也是案例分析中常用的得分点。

**【核心理论 (理解重点)】**

1. **组织文化的构成 (洋葱模型)**：
    - **物质层 (表层)**：厂容厂貌、产品外观、规章制度（易观察）。
    - **制度层 (中层)**：办事程序、行为规范。
    - **精神层 (核心)**：**价值观**、企业精神、经营哲学（最难改变）。
2. **功能**：导向、凝聚、激励、约束、辐射-。
